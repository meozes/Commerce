# Swagger API
이커머스 서비스 구현 프로젝트의 잔고, 상품, 쿠폰, 주문, 결제 API를 Swagger로 테스트하였습니다.<br>
* 테스트 DB : MySql
* 설정 정보 : application.yml
* 테스트 데이터 : data.sql
<br>
